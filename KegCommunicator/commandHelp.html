Keg Flowduino Supported Commands

  • Save to Nonvol
    SV
    Saves current system status to nonvolatile memory.
  • Set Backlight Color
    BC RRR GGG BBB
    Replace RRR/GGG/BBB with a value between 0 to 255.
  • Reset Keg Level
    RK L/R
    The L/R indicates with tap is affected.
  • Set Keg Level (Beers Left)
    SL L/R ##.#
    Sets the given keg (L or R) to the given number of beers left.
  • Set Keg Level (Pulses)
    SP L/R ####
    Sets the given keg (L or R) to the given number of pulses. This is particularly useful when combined with the Get Pulses command.
  • Get Keg Level (Pulses)
    GP L/R
    Gets the number of pulses for the given keg (L or R).
  • Set Beer Name
    SB L/R name
    Sets the given keg (L or R) to the given keg name.
  • Get Beer Name
    GB L/R
    Gets the name of the given tap (L or R).
